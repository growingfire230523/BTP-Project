# BTP-Project
Power Optimization in Railways

▪ Power Optimization in Railways | Prof. Arpan Chattopadhyay | Industry Sponsored Project [July,2024 - Nov,2024]
▪ Optimized railway equipment performance using Simplex method for linear programming, minimizing energy costs in real-time
▪ Developed algorithms integrating Q-learning , Temporal Difference learning for sequential resource allocation under uncertainty
▪ Factored "state of nature" parameters like oil temperature, motor current etc. leveraging the MetroPT-3, CWRU Bearing Datasets
▪ Ensured cost-effective energy management in operations using advanced optimization and Reinforcement Learning techniques
